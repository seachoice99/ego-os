# Cost and Token Accounting

## Goal

Ego OS must show how much each task, employee, project and model costs.

## Tracked resources

- input tokens;
- output tokens;
- cached tokens;
- reasoning tokens if provider exposes them;
- image generations;
- video generations;
- audio generations;
- API calls;
- tool execution time;
- human approval time when relevant.

## Report levels

### Task level

- total cost;
- cost by employee;
- cost by model;
- cost by tool;
- token usage;
- generated assets.

### Employee level

- cost per task;
- total cost per period;
- average quality rating;
- retry rate;
- model/tool mix.

### Project level

- total spend;
- spend by workstream;
- top expensive tasks;
- budget remaining.

### Company level

- daily, weekly, monthly spend;
- most expensive departments;
- model usage breakdown;
- cost optimization recommendations.

## CFO role

CFO monitors usage and suggests cheaper execution strategies when quality loss is acceptable.
