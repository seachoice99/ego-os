# Ego OS Product Bible

Ego OS — персональная операционная система для управления цифровой AI-компанией.

Пользователь ставит цель на естественном языке. Система сама определяет, какие цифровые сотрудники нужны, строит план, распределяет работу, ведёт отчётность, считает стоимость, сохраняет результаты и пополняет память.

## Ключевой сдвиг

Ego OS не является «чатом с агентами». Это интерфейс управления компанией, где сотрудники являются контейнерами компетенций: роль + инструкции + инструменты + доступы + модельные требования + память + отчётность.

## Принцип продукта

Пользователь не должен думать о моделях, промптах, цепочках, API и инструментах. Он должен думать о целях, проектах, качестве, сроках и бюджете.

## Основные сущности

1. Company — цифровая организация пользователя.
2. Employee — заменяемый контейнер компетенции.
3. Project — долгоживущий контекст работы.
4. Task — единица выполнения.
5. Memory — база знаний, решений, результатов и обратной связи.
6. Report — обязательная прозрачность происходящего.
7. Finance — учёт токенов, денег, времени и вычислений.

## Что важно для MVP

Первая версия должна доказать не автономность, а управляемость: пользователь видит, кто делает задачу, почему, в каком статусе, сколько это стоит и где лежит результат.
