# ADR-0003: Cost accounting is core functionality

## Status

Accepted

## Context

Complex AI tasks can consume significant tokens, API calls and generation costs. Without transparent accounting, users cannot manage budget or trust autonomous execution.

## Decision

Ego OS will track tokens, cost, tool usage and generation usage at task, employee, project and company levels.

## Consequences

- Every task must include a cost report.
- CFO role is part of the core company.
- Model selection should consider cost, quality and speed.
