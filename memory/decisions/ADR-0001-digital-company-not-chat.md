# ADR-0001: Ego OS is a digital company, not a chat

## Status

Accepted

## Context

The project could be implemented as a normal chat with multiple AI agents. This would be technically simpler but would not solve the core problem: managing long, complex intellectual work transparently.

## Decision

Ego OS will use the product metaphor of a digital company. Users will manage tasks, employees, projects, reports, memory and budgets.

## Consequences

- UI should prioritize dashboards, employees, task timelines and reports.
- Chat is only an input method.
- System entities should use company language: employee, department, project, report, finance.
