# UI Concept

## Product metaphor

LinkedIn + Jira + Mission Control for a digital AI company.

## Core screens

1. Dashboard
   - active tasks;
   - employees working now;
   - blockers;
   - today spend;
   - recent results.

2. Company
   - departments;
   - employees;
   - org structure;
   - staffing gaps.

3. Employees
   - avatar cards;
   - role profiles;
   - responsibilities;
   - current task;
   - usage and cost.

4. Tasks
   - task queue;
   - status;
   - assigned employees;
   - timeline;
   - report;
   - artifacts.

5. Projects
   - project context;
   - decisions;
   - tasks;
   - memory;
   - files.

6. Finance
   - token usage;
   - spend by model;
   - spend by employee;
   - spend by project;
   - recommendations.

7. Memory
   - decisions;
   - reusable learnings;
   - project context packs;
   - archive.

## Real-time activity

The user should see which employees are active and what each one is doing now, without exposing hidden chain-of-thought. Show operational logs only.
