# Employee Registry

## Core employees for MVP

| ID | Title | Department | Purpose |
|---|---|---|---|
| orchestrator | Orchestrator | Executive | Understand task, plan, staff, coordinate |
| pm | Project Manager | Operations | Track execution, statuses, dependencies |
| cfo | CFO | Finance | Track tokens, costs, budget, model efficiency |
| qa | QA Reviewer | Quality | Check result before delivery |
| researcher | Researcher | Research | Search, analyze, summarize evidence |
| writer | Writer | Content | Create structured texts, emails, proposals |
| designer | Designer | Creative | Visual concepts, presentation logic, image tasks |
| coder | Coder | Engineering | Code, technical implementation, repo work |

## Registry rule

Employees are versioned definitions. Updating an employee should not break old task history.
