# Ego OS

Персональная агентная операционная система.

Цель проекта — создать систему, которая умеет принимать задачи на естественном языке, выполнять их через сеть ИИ-агентов, сохранять историю работы, обучаться на обратной связи и постепенно автоматизировать всё большую часть интеллектуальной деятельности пользователя.
