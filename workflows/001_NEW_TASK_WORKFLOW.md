# New Task Workflow

1. User creates task.
2. Orchestrator creates interpretation.
3. Orchestrator estimates required capabilities.
4. Employee Registry is checked.
5. Missing employee is created/proposed if needed.
6. Plan is created.
7. PM tracks execution.
8. Employees produce outputs.
9. CFO records usage.
10. QA reviews result.
11. Final response and report are delivered.
12. Memory is updated.
